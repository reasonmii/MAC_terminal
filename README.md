### VIMRC
- .vimrc 파일 수정을 통해 vim 에디터 설정 변경 가능 (별도 플러그인 설치X)
- .vimrc 파일 만들기
  - vi ~/.vimrc
---
### Colorscheme
- vim 테마 다운받기
  - Best VIM color-schemes : https://www.slant.co/topics/480/~best-vim-color-schemes
- vim 테마 적용하기: ~/.vim/colors경로에 다운받은 테마 파일 이동
  - mkdir ~/.vim/colors
  - mv onehalfdark.vim ~/.vim/colors/
- ~/.vimrc 파일 수정
  - vi ~/.vimrc
  - vimrc 파일 참고 : ctrl + v
