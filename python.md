terminal package 설치
```vim
sudo pip3 install <package>
sudo pip3 install <package> --upgrade
```

python 실행
```vim
python3
```

python 파일 생성 및 작업
```vim
vi fileNmae.py
i                    // python script 입력
esc + :wq            // 저장+종료
ls                   // 파일확인
python3 fileName.py  // 파일 내용 확인
```
  
jupyter 작업하기
```vim
jupyter notebook     // jupyter 실행
ctrl + C : shutdown
// jupyter notebook 사용 중 terminal에서 jupyter notebook 종료하기
```
